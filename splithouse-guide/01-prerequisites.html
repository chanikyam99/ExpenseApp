Part 01

Prerequisites

Install all the tools you need before writing a single line of code. This takes about 15 minutes.

Step-by-step install

1
Install Node.js 20 LTS

Node.js is the runtime that powers Next.js. You need version 20 (LTS = Long Term Support = most stable).

  1. Go to nodejs.org
  2. Click the big green "20.x LTS" button (not Current)
  3. Download the Windows installer (.msi)
  4. Run it — accept all defaults, make sure "Add to PATH" is checked
⚠️
Do not install Node 22 or 23 — they are "Current" releases and some packages are not yet tested against them. Stick with 20 LTS.

After installing, open a new terminal (PowerShell or Command Prompt) and verify:

Terminal

You should see something like v20.18.0 and 10.8.2.

2
Install Git

Git lets you version your code and connect it to GitHub for deployment on Vercel.

  1. Go to git-scm.com/download/win
  2. Download the 64-bit installer
  3. Run it — all defaults are fine. On the "Adjusting your PATH" screen choose "Git from the command line and also from 3rd-party software"

Verify in a new terminal:

Terminal

You should see something like git version 2.47.0.windows.1.

3
Install Visual Studio Code

VS Code is the code editor. You may already have it since you're reading this in VS Code.

  1. Go to code.visualstudio.com
  2. Download and install for Windows
4
Install VS Code Extensions

Open VS Code → press Ctrl+Shift+X → search and install each:

ExtensionWhat it does
ESLintShows code errors as you type
Tailwind CSS IntelliSenseAutocomplete for Tailwind class names
Prettier - Code formatterAuto-formats code on save
TypeScript ImporterAuto-imports TypeScript modules
💡
To enable Prettier on save: press Ctrl+Shift+P → type "Open User Settings JSON" → add "editor.formatOnSave": true
5
Create a GitHub account

GitHub stores your code and is what Vercel connects to for automatic deployments.

  1. Go to github.com → Sign up (free)
  2. Verify your email

Then configure Git with your identity (one-time setup):

Terminal
6
Create a Vercel account

Vercel hosts your app for free. Sign up at vercel.com using your GitHub account — this links them together for one-click deploys later.

✅
All done with prerequisites! You have Node.js, Git, VS Code, GitHub, and Vercel ready.
